🎓 Project Features

✅ Lexical Analysis

Tokenizes SQL keywords, identifiers, operators, and literals using Flex

✅ Syntax Analysis

Validates SQL grammar with LALR parser using Bison

✅ Query Execution

Processes CSV files and evaluates WHERE conditions

✅ Boolean Logic

Supports AND/OR operators with proper precedence
